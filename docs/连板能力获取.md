根据您提供的知识库资料，通过量化数据接口获取历史连板天梯数据（SC23）的具体使用方法如下：
一、 核心接口与字段说明

数据编号：SC23
数据项：连板家数
输出字段：

第1个数据：连板股个数（包含ST和未开板新股）
第2个数据：连板股个数（不含ST股和未开板新股）

历史数据范围：仅支持 2018年3月19日（20180319） 之后的数据。

二、 TdxQuant Python 接口调用方法
在通达信量化平台（TdxQuant）中，您可以通过 tq.formula_set_data_info 配合 tq.formula_get_data 来获取该数据。具体步骤与代码如下：
步骤 1：设置并获取指定历史时间段的K线数据
使用 formula_set_data_info 设定要查询的股票（通常取大盘指数如上证指数来代表市场）、周期及历史时间段。
from tqcenter import tq

# 初始化
tq.initialize(__file__)

# 设置数据信息（以获取2025年11月的数据为例）
# 注意：stock_code 一般使用大盘指数代码（如 000001.SH），period设为日线
formula_set_res = tq.formula_set_data_info(
    stock_code='000001.SH', 
    stock_period='1d', 
    start_time='20251101',  # 历史起始时间
    end_time='20251130',    # 历史结束时间
    count=0,                # count为0时，start_time和end_time生效
    dividend_type=0         # 不复权
)
print(formula_set_res)
# 预期输出: {'ErrorId': '0', 'Msg': '向通达信公式系统设置数据信息成功！', 'run_id': '1'}
步骤 2：在公式中调用 SC23 并获取计算结果
在通达信公式中，市场级别的专业财务数据通过 FINVALUE 函数调用。SC23的第1个数据为 FINVALUE(23, 1)，第2个数据为 FINVALUE(23, 2)。
您可以在自定义公式中写入这些代码，然后使用 formula_get_data 获取结果。
# 获取目前公式设置中的K线数据及计算结果
# （需要先在通达信客户端下载对应的盘后数据）
result_df = tq.formula_get_data()
print(result_df)
(注：formula_get_data 会返回包含您在公式中计算的所有数据列的 DataFrame，其中就包含了指定时间段内每天的 SC23 连板家数统计)

三、 在通达信客户端公式中的直接用法
如果您是在通达信客户端内编写指标公式或选股公式，可以直接使用 FINVALUE 函数：
// 获取市场连板家数（包含ST和未开板新股）
连板家数1: FINVALUE(23, 1);

// 获取市场连板家数（不含ST股和未开板新股）
连板家数2: FINVALUE(23, 2);
⚠️ 使用注意事项

盘后数据依赖：知识库特别强调，获取历史K线数据需要先在客户端中下载对应的盘后数据，否则调用会失败或触发长时间刷新。
参数设置冲突：在使用 formula_set_data_info 时，count 参数最大值为24000；当 count 不为0时（如count=100），start_time 和 end_time 会失效，系统会截取最新交易日开始往前的n条K线。因此，获取特定历史时间段务必将 count 设为 0。
数据覆盖范围：切记 SC23 仅展示 2018年3月19日之后的数据，调用更早的日期将返回空值。
