## 二、二进制文件格式

### 2.1 K线文件 (tdx_reader.py)

| 文件类型 | 格式 | 字段 | 价格处理 | 路径 |
|---------|------|------|---------|------|
| `.day` 日线 | 32B/条 | date, open, high, low, close, amount, volume | u32 ÷100 | `sh/sz/lday/` |
| `.lc1` 1分钟 | 32B/条 | date_num, minutes, open, high, low, close, amount, volume | f32 直接 | `sh/sz/minline/` |
| `.lc5` 5分钟 | 32B/条 | 同上 | f32 直接 | `sh/sz/fzline/` |

**日期编码**:
```python
# .day: YYYYMMDD 整数
year = date // 10000

# .lc1/.lc5: 特殊编码
year = date_num // 2048 + 2004
month = (date_num % 2048) // 100
day = (date_num % 2048) % 100
hour = minutes // 60
minute = minutes % 60
```

### 2.2 股票基础数据 (base.dbf)

**DBF文件结构**:
- 头32字节: 版本、日期、记录数
- 字段定义: 40个字段
- 数据区: 每条记录481字节

**关键字段**:
| 字段名 | 长度 | 含义 |
|--------|------|------|
| GPDM | 6B | 股票代码 |
| GXRQ | 8B | 除权日期 |
| ZGB | 14B | 总股本 (万股) |
| LTAG | 14B | 流通A股 (万股) |
| JZC | 14B | 净资产 (元) |
| ZZC | 14B | 总资产 (元) |
| HY | 4B | 行业代码 |
| SSDATE | 8B | 上市日期 |
| GDRS | 8B | 股东人数 |

## 三、tdx_reader.py 能力清单

### 3.1 K线读取方法

```python
from tdx_reader import TdxReader

reader = TdxReader(n_workers=4)  # n_workers: 并行进程数

# 串行流式读取（内存友好）
for df in reader.read_daily(batch_size=500):
    process(df)
```

## 四、股票代码过滤规则

```python
def is_stock(code: str) -> bool:
    """判断是否为A股"""
    return code[:3] in ('000', '001', '002', '003', '300', '600', '601', '603', '688')

def is_etf(code: str) -> bool:
    """判断是否为ETF"""
    return code[:3] in ('510', '511', '512', '513', '515', '159')

def is_cb(code: str) -> bool:
    """判断是否为可转债"""
    return code[:3] in ('110', '113', '127', '128')

def is_index(code: str) -> bool:
    """判断是否为指数"""
    return code in {
        '000001.SH', '399001.SZ', '399006.SZ',
        '000300.SH', '000905.SH', '000852.SH',
        '000016.SH', '399673.SZ', '000688.SH',
    }

def get_market_suffix(code: str) -> str:
    """获取市场后缀"""
    if code.startswith(('6', '5', '9')):
        return '.SH'
    elif code.startswith(('0', '1', '2', '3', '8')):
        return '.SZ'
    elif code.startswith('4'):
        return '.BJ'
    return ''
```

## 五、性能优化要点

### 5.1 已实现优化

| 优化手段 | 效果 | 实现方式 |
|---------|------|---------|
| numpy.fromfile() | 10-50x | 批量读取整个文件 |
| numpy 向量化日期解码 | 5-10x | 数组运算代替循环 |
| 多进程并行 | 2-4x | ProcessPoolExecutor |
| 时间范围过滤 | 50%+ | min_date/max_date 参数 |
| COPY + Parquet 导入 | 5-10x | 最优方案，内存稳定 |
| 逐文件处理 | 内存稳定 | 避免全量加载 |

### 5.2 COPY 导入方案（最优）

```python
import tempfile
import os

def save_data(con, df):
    """用 COPY + Parquet 导入，比 INSERT 快 5-10 倍"""
    with tempfile.NamedTemporaryFile(suffix='.parquet', delete=False) as tmp:
        parquet_path = tmp.name
    try:
        df.to_parquet(parquet_path, index=False)
        con.execute(f"COPY {TABLE} FROM '{parquet_path}' (FORMAT PARQUET)")
    finally:
        if os.path.exists(parquet_path):
            os.unlink(parquet_path)
```
