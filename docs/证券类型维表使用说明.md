## 三、表结构

```sql
CREATE TABLE dim_security_type (
    code        VARCHAR PRIMARY KEY,    -- 证券代码
    type        VARCHAR,                -- 类型
    market      VARCHAR,                -- 市场(SH/SZ)
    prefix      VARCHAR,                -- 前缀
    is_active   BOOLEAN DEFAULT TRUE,   -- 是否活跃
    created_at  TIMESTAMP,               -- 创建时间
    updated_at  TIMESTAMP               -- 更新时间
);
```

## 四、使用示例

### 1. 查询所有ETF

```sql
SELECT code FROM dim_security_type WHERE type = 'ETF';
```

### 2. 查询所有板块

```sql
SELECT code FROM dim_security_type WHERE type = '板块';
```

### 3. 查询沪市主板股票

```sql
SELECT code FROM dim_security_type WHERE type = '沪市主板';
```

### 4. 查询可转债

```sql
SELECT code FROM dim_security_type WHERE type = '可转债';
```

### 5. 按类型统计K线数据

```sql
-- 统计各类型证券在5m K线中的数据量
SELECT 
    d.type,
    COUNT(*) as kline_count,
    COUNT(DISTINCT d.code) as stock_count
FROM dim_security_type d
JOIN stock_kline_5m k ON d.code = k.code
GROUP BY d.type
ORDER BY kline_count DESC;
```

### 6. 查看某只证券类型

```sql
SELECT type, market FROM dim_security_type WHERE code = '600519.SH';
```

## 七、注意事项

1. **数据来源**: 维表数据从 `stock_kline_*` 表自动提取
2. **增量更新**: 新增证券会自动同步到维表
3. **类型修正**: 运行同步脚本会自动修正所有证券类型
4. **分类依据**: 纯基于 code 前缀规则，非实时财务数据
