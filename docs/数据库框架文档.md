## 二、ODS 层（原始数据层）

直接从外部数据源获取的原始数据。

> **注意**: 指数/ETF/可转债/板块的K线数据统一存储在 `stock_daily_kline` / `stock_kline_*` 表中，通过 `dim_security_type` 区分类型。

| 表名 | 中文名 | 数据源 | 说明 |
|------|--------|--------|------|
| `stock_daily_kline` | 股票日K线 | TDX .day | A股日线（含指数/ETF/可转债/板块）|
| `stock_kline_1m` | 股票1分钟K | TDX .lc1 | 分钟K线（含指数/ETF）|
| `stock_kline_5m` | 股票5分钟K | TDX .lc5 | 分钟K线（含指数/ETF）|
| `etf_daily_kline` | ETF日K线 | TDX .day | ETF日K线（独立表）|
| `sector_daily_data` | 板块日行情 | TDX .day | 板块日K线 |
| `stock_trading_data` | 股票交易数据 | TQ API | GPJY |
| `market_trading_data` | 市场交易数据 | TQ API | SCJY |
| `fact_finance_report` | 财务报告 | TDX gpcw | 原始财务 |
| `trading_calendar` | 交易日历 | TQ API | 交易日 |

> ~~`index_daily_kline`~~ → 已合并到 `stock_daily_kline`
> ~~`index_kline_1m/5m`~~ → 已合并到 `stock_kline_1m/5m`
> ~~`etf_kline_1m/5m`~~ → 已合并到 `stock_kline_1m/5m`
> ~~`cb_daily_kline`~~ → 已合并到 `stock_daily_kline`
> ~~`sector_kline_daily/5m`~~ → 已合并到 `sector_daily_data` / `stock_kline_5m`

## 三、DWD 层（明细宽表）

对 ODS 层数据进行清洗、标准化。

| 表名 | 中文名 | 来源 | 说明 |
|------|--------|------|------|
| `dwd_stock_capital_flow` | 资金流向 | ODS | 标准化资金流 |
| `dwd_stock_intraday_feature` | 日内特征 | ODS:kline_1m | 分钟特征 |
| `dwd_stock_limit_up_feature` | 涨停特征 | ODS | 涨停统计 |
| `stock_basic_info` | 股票基本信息 | TDX | 基本信息 |
| `stock_capital_data` | 股本数据 | TDX base.dbf | 股本信息 |
| `stock_sector_relation` | 股票板块关系 | TDX | 板块归属 |
| `technical_indicators` | 技术指标 | TQ API | MACD/KDJ等 |

## 九、数据流向图

```
                    ┌──────────────┐
                    │   TDX文件    │
                    │ .day/.lc1/.lc5
                    └──────┬───────┘
                           │ numpy
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    ODS 层（原始数据）                      │
│  stock_daily_kline  ← 股票+指数+ETF+可转债+板块 日K       │
│  stock_kline_1m/5m  ← 股票+指数+ETF+板块 分钟K           │
│  etf_daily_kline    ← ETF日K（独立表）                   │
│  sector_daily_data  ← 板块日行情                         │
│  stock_trading_data / market_trading_data (TQ API)     │
└────────────────────────────┬──────────────────────────────┘
                           │
          ┌────────────────┼────────────────┐
          ▼                ▼                ▼
┌─────────────────┐ ┌──────────────┐ ┌──────────────┐
│   DWD 层        │ │  DWS 层     │ │   DM 层      │
│ (清洗标准化)     │ │ (汇总)      │ │ (聚合)       │
│ stock_basic_info│ │ dws_sector  │ │ stock_weekly │
│ stock_capital   │ │ stock_extend│ │ kline_15m   │
│ stock_sector   │ │ etf_derived │ │ kline_30m   │
│ technical_ind   │ └──────────────┘ │ kline_60m   │
└─────────────────┘                  └──────────────┘

统一通过 dim_security_type 区分：股票/指数/ETF/可转债/板块
```
