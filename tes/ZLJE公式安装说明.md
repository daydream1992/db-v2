# ZLJE 主力净额公式 — 完整建立流程

> 目标:在通达信里建一个名为 `ZLJE` 的自定义技术指标公式,
> 让 TQ 能通过 `tq.formula_process_mul_zb(formula_name='ZLJE')` 调到历史主力净额数据。
>
> **为什么必须建**: `get_more_info` 的 `Zjl` 字段是**盘中实时数据,盘后归零**(已实测),
> 无法用于盘后选股/回测。ZLJE 基于 K 线的 `L2_AMO` 函数,能取**历史区间**,盘后也有值。

---

## ✅ 前置确认(建之前先看)

1. **已登录通达信交易账号** —— 公式存在本地用户配置里,游客模式可能无法保存
2. **已订阅 Level-2 数据** —— `L2_AMO` 是 L2 专属函数(你说已订阅,OK)
3. **客户端是专业研究版 V7.73+ 或金融终端 V7.72+**(带 TdxQuant 的版本,你的有 `PYPlugins` 目录,符合)

## 📝 第四步:填写公式属性(顶部栏位)

在编辑器顶部填:

| 栏位 | 填什么 | 注意 |
|------|--------|------|
| **公式名称** | `ZLJE` | ⚠️ **必须全大写**,跟脚本 `formula_name='ZLJE'` 一致 |
| **公式描述** | `主力净额` | 随便填,备注 |
| **密码保护** | 留空 | 不加密 |
| **画线方法** | 副图 | 点下拉选「副图」(默认主图也行,反正不画线) |
| **参数表** | 留空 | 公式没参数,不用填 L1/N1 之类 |
| **坐标线** | 留空 | 不用 |
| **额外功能** | 不勾 | 不用 |
| **用法说明** | 留空 | 可选 |

## 📝 第五步:粘贴公式源码(最关键)

把下面这段**一字不差**地复制,粘进编辑器中间的**大代码输入框**:

```
{ZLJE - 主力净额指标}
超B:=L2_AMO(0,0)/10000.0;
大B:=L2_AMO(1,0)/10000.0;
中B:=L2_AMO(2,0)/10000.0;
小B:=L2_AMO(3,0)/10000.0;

超S:=L2_AMO(0,1)/10000.0;
大S:=L2_AMO(1,1)/10000.0;
中S:=L2_AMO(2,1)/10000.0;
小S:=L2_AMO(3,1)/10000.0;

主力净额:(超B+大B)-(超S+大S),NODRAW;
```

### 源码逐行解释(理解了以后好排错)

- `L2_AMO(档位, 方向)` 是 Level-2 成交额函数,单位**元**
  - 第1参数 档位: `0`=超大单, `1`=大单, `2`=中单, `3`=小单
  - 第2参数 方向: `0`=买入, `1`=卖出
  - `/10000.0` 把元换成**万元**
- `超B/大B/中B/小B` = 各档**买入**额(万元)
- `超S/大S/中S/小S` = 各档**卖出**额(万元)
- `主力净额 = (超大单+大单)买入 − (超大单+大单)卖出` —— 即主力资金净流入
- `:=` 是中间变量(不输出), 最后一句不带 `:` 的才是输出值
- `NODRAW` = 只算值不画线(纯数据用,我们要的是值不是图)

## 🛠 常见报错排查

### 报错 1:`L2_AMO 是未知函数` / `未知字符串`
**原因**: 客户端不支持 Level-2,或 L2 权限未生效。
**解决**:
- 确认是专业研究版 V7.73+
- 重新登录通达信账号(让 L2 权限刷新)
- 还不行就是版本太旧,去 tdx.com.cn 下最新专业研究版

### 报错 2:`获取公式失败或公式不存在`(跑脚本时)
**原因**: 公式没建成功 / 名字不是 `ZLJE` / 建在错的分类下。
**解决**:
- 回公式管理器确认「技术指标公式」下有 `ZLJE` 这一项
- 名字必须全大写 `ZLJE`,不能是 `Zlj`/`zlje`/`主力净额`
- 必须建在「**技术指标公式**」分类,不是条件选股/交易系统

### 报错 3:`语法错误` / `缺少分号`(测试公式时)
**原因**: 粘贴时漏了符号。
**解决**: 每行结尾必须有 `;`(最后一句 `主力净额:...;` 也要分号),`:=` 和 `:` 别搞混。

### 报错 4:测试通过但脚本跑出全是 0 或空
**原因**: L2 历史数据该日没下载,或非交易时段无实时数据。
**解决**:
- 历史区间换近的交易日(脚本 `--start/--end`)
- 确认通达信里这些股票的 L2 数据已下载(系统选项 → 数据下载)
