# ZLJE 主力净额公式 — 安装说明

> `tes_011_zlje.py` 跑之前**必读必做**。ZLJE 不是通达信内置公式,
> 是用户自定义公式,必须先在客户端手动建好,TQ 才能通过 `formula_process_mul_zb('ZLJE')` 调到。
> 否则报错:**获取公式失败或公式不存在**(我们 tes_004 已实测复现)。

## 一、公式源码(直接复制)

```
{ZLJE - 主力净额指标}
超B:=L2_AMO(0,0)/10000.0;  {超大单买入}
大B:=L2_AMO(1,0)/10000.0;  {大单买入}
中B:=L2_AMO(2,0)/10000.0;  {中单买入}
小B:=L2_AMO(3,0)/10000.0;  {小单买入}

超S:=L2_AMO(0,1)/10000.0;  {超大单卖出}
大S:=L2_AMO(1,1)/10000.0;  {大单卖出}
中S:=L2_AMO(2,1)/10000.0;  {中单卖出}
小S:=L2_AMO(3,1)/10000.0;  {小单卖出}

主力净额:(超B+大B)-(超S+大S),NODRAW;
```

**字段说明:**
- `L2_AMO(档位, 方向)` — Level-2 成交额函数
  - 档位: `0`=超大单 / `1`=大单 / `2`=中单 / `3`=小单
  - 方向: `0`=买入 / `1`=卖出
  - 除以 10000 是把单位从元换成万元
- **主力净额 = (超大单+大单)买入额 − (超大单+大单)卖出额**
- `NODRAW` 表示只算值不画线(纯数据指标)

## 二、安装步骤

1. 打开通达信客户端,快捷键 `Ctrl+F` 打开**公式管理器**
   (或菜单:功能 → 公式系统 → 公式管理器)
2. 左侧树选 **技术指标公式** → 点 **新建**
3. 填:
   - **公式名称**:`ZLJE`(必须全大写,跟脚本里一致)
   - **画线方法**:副图(随便,反正 NODRAW 不画)
4. 把上面的源码粘进编辑区
5. 点 **保存** → 关闭公式管理器

## 三、权限要求

`L2_AMO` 是 **Level-2 行情函数**,需要:
- 通达信**专业研究版 V7.73(64位)** 或 **金融终端(量化模拟) V7.72**
- 已订阅 **Level-2 数据**(资金流向数据是 L2 专属)

普通免费版 `L2_AMO` 会返回空,脚本算出来全是 0 差额。

## 四、验证

建好公式后跑:
```bash
python K:/DB数据库_v2/tes/tes_011_zlje.py --codes 600519.SH,000001.SZ --rounds 2 --interval 5
```
- 看到 `🔥 主力净额变化前 5 名` 列表 → 公式建对了
- 还报 `获取公式失败或公式不存在` → 回第一步检查公式名是否 `ZLJE`、是否保存成功
- 列表是空的/全 0 → 多半是 L2 数据没权限或非交易时段
