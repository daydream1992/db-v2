# DataOps 管理台 · 使用说明

## 打开

双击 `dataops-ui\index.html`，浏览器直接打开，无需端口/服务器。

> Chrome 如页面空白：在 `dataops-ui\` 目录执行 `python -m http.server 8080`，浏览器访问 `http://localhost:8080`
>
> 或直接用 **Edge** 打开 index.html，无此问题。

## 10 个视图

| 视图 | 功能 |
|------|------|
| Dashboard | KPI 趋势、告警总览、7/30/90天切换、KPI 钻取 |
| 脚本目录 | 26 表注册中心，搜索表名/脚本名 |
| 健康度 | 红绿灯矩阵 + 交易日历校验 + 异常归因 |
| 编排 | DAG 依赖 + Gantt 时序 + 执行详情 |
| 血缘 | 表↔脚本关系图，拖拽/minimap/焦点高亮 |
| 规范校验 | 12 条规则违规热力图 |
| 日志 | 按表/级别/执行分组，导出 .log |
| 数据字典 | Schema Diff + 字段搜索 + Markdown 导出 |
| SQL Playground | 多 Tab + 结果对比 + 语法高亮 + CSV 导出 |
| 设置 | Lint 规则/调度/通知/YAML 导入导出 |

## 快捷键

| 键 | 功能 |
|----|------|
| Ctrl+K | 命令面板（全局搜索） |
| ? | 快捷键帮助 |
| Ctrl+T / Ctrl+W | SQL 新建/关闭 Tab |
| Ctrl+Enter | SQL 执行 |
| Ctrl+S | 设置页保存 |

## 交易日历校验

健康度视图的日期检测**查询 `trading_calendar` 表的 `is_trading` 字段**：
- 数据源：`SELECT date, is_trading FROM trading_calendar`，非脚本执行日志推断
- 只校对 `is_trading=true` 的日期，周末/节假日自动跳过
- 非交易日矩阵单元格显示 `–`（跳过），趋势图显示"休"
